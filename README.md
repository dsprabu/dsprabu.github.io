# DSPRABU Personal Website

## Features
- Modern colorful animated design
- Photography gallery
- Tamil stories section
- Mobile responsive
- GitHub Pages ready
- Basic image protection

## Setup
1. Upload all files to:
   dsprabu.github.io repository

2. Go to:
   Settings → Pages

3. Enable:
   Deploy from branch → main

4. Your site will be live at:
   https://dsprabu.github.io

## Add Your Images
Replace images inside:
images/

Recommended:
- profile.jpg
- photo1.jpg
- photo2.jpg
- photo3.jpg
